📺 Comment utiliser avec OBS Studio

VersetLive génère une page d'affichage transparente que vous pouvez ajouter comme source navigateur dans OBS.

Étape 1 — Ajouter une source navigateur dans OBS

  1. Dans OBS Studio, cliquez sur + sous "Sources"
  2. Choisissez "Source du navigateur"
  3. Donnez un nom (ex: Versets) et validez

Étape 2 — Configurer l'URL

Cliquez sur "Ouvrir vue OBS" en haut à droite : la page d'affichage s'ouvre. Copiez l'URL complète depuis la barre d'adresse de votre navigateur, puis collez-la dans le champ URL d'OBS.

Réglages recommandés dans OBS :

  • Largeur : 1920
  • Hauteur : 1080
  • Cochez "Arrière-plan transparent" (CSS personnalisé : body { background: transparent !important; })
  • Cochez "Rafraîchir le navigateur lorsque la scène devient active"

Étape 3 — Diffuser un verset

  1. Sélectionnez un verset dans le panneau gauche
  2. Cliquez sur ▶ Diffuser (ou double-clic sur un verset)
  3. Le verset apparaît instantanément dans OBS et sur votre flux

Astuce

La page OBS et ce panneau de contrôle se synchronisent automatiquement via BroadcastChannel et localStorage. Tant que les deux pages sont sur la même origine (même URL de base), tout est instantané.

Hors ligne

L'onglet Manuel permet de saisir n'importe quel verset à la main si vous n'avez pas accès à internet (l'API bolls.life est utilisée pour récupérer les textes en ligne).
